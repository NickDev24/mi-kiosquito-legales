# Mi Kiosquito - Documentos Legales

Este repositorio contiene los documentos legales de la aplicación móvil **Mi Kiosquito**, diseñada para la gestión de pequeños negocios en Argentina.

## 📋 Sobre este Proyecto

Este repositorio aloja la documentación legal requerida por **Google Play Console** para la publicación de la aplicación Mi Kiosquito. Los documentos están redactados conforme a la legislación argentina y están disponibles públicamente a través de GitHub Pages.

## 📱 ¿Qué es Mi Kiosquito?

**Mi Kiosquito** es una aplicación móvil para Android que permite a pequeños comercios (kioscos, despensas, almacenes, etc.) gestionar:

- **Inventario**: Control de productos, stock y códigos de barras
- **Caja diaria**: Registro de ventas, gastos y balance
- **Clientes**: Base de datos de clientes y su historial
- **Fiados**: Control de cuentas por cobrar y deudas
- **Reportes**: Estadísticas básicas de ventas y movimientos

### Planes Disponibles

- **Plan Gratis**: Funcionalidades básicas con publicidad
- **Plan Premium**: Funcionalidades avanzadas, exportación de datos, soporte prioritario y sin publicidad
